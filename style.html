Styling HTML Elements

Closing Tags

All tags that 'wrap' stuff (like text or other tags) must be closed, otherwise odd thigs can happen!

The 'closing' tag of an element always has a '/' before the letter(s).

Self Closing Tags

Some elements don't have a closing tag as they don't wrap anying.
Like the line break tag, which tells text to break to the
next line.

Element Attributes

Tags can also contain attributes. Attributes provide extra information about HTML elements.

Some elements have required attributes. Like the image 'src' and 'alt' tags.
